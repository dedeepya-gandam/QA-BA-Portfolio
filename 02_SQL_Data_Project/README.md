# SQL Data Analysis Project — Retail Sales Dataset

This project demonstrates SQL skills including data cleaning, transformations, aggregations, joins, window functions, CTEs, subqueries, and performance optimization.

## Dataset
A sample retail sales dataset with:
- Orders
- Customers
- Products
- Sales Transactions

## Key SQL Concepts Demonstrated
- Joins (INNER, LEFT, RIGHT)
- Filtering & sorting
- Aggregations (SUM, AVG, COUNT)
- GROUP BY & HAVING
- Window functions (ROW_NUMBER, RANK, PARTITION BY)
- Common Table Expressions (CTE)
- Subqueries (correlated + non-correlated)
- Data cleaning using TRIM(), COALESCE(), CAST(), NULL handling
- Creating views
- Index performance check

## Example Queries Included
- “Top 10 customers by purchase amount”
- “Monthly sales trend”
- “Orders with missing customer details”
- “Repeat purchase analysis”
- “Category-wise revenue contribution”
- “Customer churn identification”
